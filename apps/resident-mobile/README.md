# EkoTrust Mobile

Android-first Flutter app for EkoTrust, the reputation operating system for
Africa's informal economy.

The current MVP focuses on:

- Identity verification with phone, liveness, community, and profile steps.
- Visual Proof of Work capture for before/after job evidence.
- Peer and community reputation signals.
- Trust score, verification level, and public shareable artisan profile.
- Low-end Android performance with simple native Flutter rendering.

Run locally:

```powershell
flutter pub get
flutter test
flutter build apk --debug
```
