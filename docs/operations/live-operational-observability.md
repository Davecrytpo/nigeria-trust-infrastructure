# Live Operational Observability

Status: Pilot observability baseline.

## Required Visibility

- Active degraded states.
- Operator cognitive load.
- Telecom instability.
- Replay divergence.
- Trust volatility.
- Institutional non-response.
- Escalation bottlenecks.
- Recovery convergence health.

## Pilot Rule

If operators cannot see degradation, the pilot must not run live. Blind operation converts uncertainty into unmanaged civic risk.

## Watch Duties

- Monitor active degraded modes.
- Watch operator load and unresolved critical incidents.
- Flag telecom instability and delayed receipts.
- Track replay divergence before closure.
- Track trust volatility and responder anomalies.
- Record institutional non-response.
- Confirm recovery convergence before normalizing operations.
